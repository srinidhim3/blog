A couple of months back when I started learning more about machine learning from Udemy courses, which covered most of the basic intuition about statistics and whatnot. But then I started discussing my learning progress with a colleague. He told me that deep learning gives good results in most cases when compared to regular machine learning models.

Until this point, I had no clue about what is deep learning or anything related to it. As I was going through a course from Udemy I thought let me complete this course and then start exploring more on deep learning. Thankfully, that particular course had the last section dedicated to deep learning.

Though the last section did not cover many details, it was a fair amount of intro to deep learning. Then I discovered that there is a module called Tensorflow by Google for deep learning applications. From that point onwards I've concentrated more on Tensorflow rather than any other module. 

I've taken up a couple of courses from Coursera on Tensorflow but later on, got to know that there is Tensorflow developer certification provided by Google. Hence currently preparing for the certification. Below are the courses I've taken on machine learning and deep learning. Let me know if you find the courses helpful.
